# learn-rust-with-cli

Use the way of developing command-line applications to practice and learn the rust language at the same time.

用开发命令行应用程序的方式，一边实践一边学Rust语言。

# Run & Build
```
# Run
cargo run rust-by-example:hello 
# Build
cargo run rust-by-example:hello 
```
